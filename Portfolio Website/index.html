For HTML, CSS and Javascript, I learnt that it is used to create websites. I learnt about HTML which stands for Hyper Text Markup Language and it is a code that allows us to build websites. The anatomy of a HTML element is the building blocks of web pages and examples of this would be paragraph, heading, div, list etc. It uses tags such as opening tags which mark the beginning of an element and a closing tag marks the end. I learnt about CSS which stands for Cascading Style Sheets and it is not a programming language but it is a style sheet language that lets us style the elements on my page. It works in conjunction with HTML but it is not HTML itself. CSS associates its style rules with HTML elements and CSS properties affect how elements are displayed. I learnt about Javascript which can make websites more interactive, interesting and user-friendly. It lets you access and modify the content and markup of web pages as the user is viewing them. I have done two CAs which involve creating a website. The first CA was about building a website using the front-end technologies HTML, CSS and JavaScript. I have also experience in cloud computing during my 3 years in my Bachelors in Computing and I.T course. We mostly used Google Cloud. I learnt about cloud which is a distinct IT environment that is designed for the purpose of remotely provisioning scalable and measured IT resources. It is privately owned and offers access to IT resources that are metered. Clouds can be based on the use of any protocols that allow for the remote access to its IT resources. I learnt about scaling which represents the ability of the IT resource to handle increased or decreased usage demands. Two types of scaling are horizontal scaling which scales out and scales in while vertical scaling which scales up and scales down. I learnt about setting up different types of virtual machines on Google cloud which allows us to use different operating systems such as Windows and Linux. I also learnt about setting up storage buckets on Google Cloud which allows us to store and upload files into the storage buckets. I can also create new folders and delete existing files and folders. I learnt about downloading Google SDK which is a set of tools and it contains gcloud, gsutil and bq command-line tools which you can use to access Google Compute Engine, Google Cloud Storage, Google BigQuery and other products and services from the command line. I have done 2 CAs on cloud computing. The first CA involves researching on a data centre, writing about the cost of equipment, security restrictions and discussing three main service models of cloud computing. The second CA involves using Google Cloud and I have to autoscale, stress test, monitor and network load balancing for a test website.
